PrivyPrompt

Privacy policy for your local AI privacy firewall.

Last updated: July 7, 2026

PrivyPrompt helps users review sensitive information and risky AI-tool instructions before they submit prompts or supported files to generative AI websites.

Local processing

  • Prompt text and supported file contents are processed locally inside the user's browser before submission.
  • AI tool security checks are local pattern checks for prompt-injection, jailbreak, system-prompt leakage, local-secret access, and secret-exfiltration wording.
  • PDF.js, Mammoth, Tesseract.js, WebAssembly files, and OCR data are bundled with the extension and do not load remote code at runtime.

What PrivyPrompt does not send to the developer

No prompt uploads No telemetry No advertising data

PrivyPrompt does not transmit prompts, supported files, OCR text, detected values, custom protected terms, private aliases, or AI responses to the developer or advertising services.

Local storage

PrivyPrompt uses browser extension storage for protection preferences, selected country packs, user-created protected terms, and aggregate counters such as local scan and detection counts. The extension does not intentionally save complete prompts or detected sensitive values.

Supported AI websites

Website access is limited to explicitly supported AI services so PrivyPrompt can inspect user-initiated prompts and file selections immediately before submission and display local protection controls. The destination AI service receives only the content that the user approves and sends through that service.

Private aliases and response restoration

When a user chooses to protect selected values, PrivyPrompt creates session-only placeholders. Placeholder-to-original mappings exist only in the current tab's extension memory and disappear when the page closes or reloads. Restored originals are displayed in a closed extension-created view and are not inserted into the AI site's normal readable page DOM.

Network activity

PrivyPrompt does not make external network requests for analytics, advertising, prompt processing, document processing, OCR, or AI review. Any future feature involving remote processing will require clear disclosure and user consent before activation.

Data selling and advertising

PrivyPrompt does not sell user data and does not use user data for advertising.

Changes and contact

This policy will be updated if PrivyPrompt's data handling changes. For support or privacy questions, use the support contact provided on PrivyPrompt's store listing or product page.
